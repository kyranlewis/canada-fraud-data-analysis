# Canada Fraud Data Analysis

For a final project in another Python class, my team was tasked with performing data analysis on data imported from an open data source. The data source my partner chose came from the Open Government Canada website, via their Open Data Lake.

Within this huge data source contained many different datasets that you could pick and choose from to create analysis upon, what I chose to do my analytics on was a fraud dataset within the data lake.


Originally, the data inside of the dataset that I chose was primarily in french, so I had to utilize AI to completely translate all columns for myself to make it readable in english before I could begin any kind of analysis on the dataset.
